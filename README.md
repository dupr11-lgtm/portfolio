# duparc.studio

Welcome to **Alienor**! This site is a rebranded take on the original project, tailored to highlight the duparc.studio domain while keeping the same mission: making educational content available to everyone, no matter where they are.

## About the Project

duparc.studio is a community-driven hub where contributors share deep-dives on programming, tooling, and workflow tips. The content lives in `src/content/posts`, and each post generates a static page through Astro.

## Getting Started Locally

1. Install [bun](https://bun.sh/) (if you haven’t already).
2. Run `bun install` to sync the dependencies.
3. Launch the dev server with `bun dev` and open `http://localhost:3000` in your browser.
